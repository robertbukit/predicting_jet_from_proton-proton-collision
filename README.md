Project ini bertujuan untuk memprediksi jumlah keluaran partikel jet yang dihasilkan oleh peristiwa tumbukan antar proton menggunalan model machine learning. 
Data ini didapat dari percobaan/eksperimen LHC di CERN. pada tahun 2010.

# Konten Repositori
1. nJet_predicted_from_proton_proton_collision.ipynb
2. MultiJetRun.csv
3. Jet Particle.pdf

# Model yang dipakai
1. Random Forest
2. K-Nearest-Neighbors (KNN)
3. XGBoost

# Peralatan
1. Python 3.8
2. skicit-learn
3. numpy
4. pandas
5. matplotlib
6. seaborn
